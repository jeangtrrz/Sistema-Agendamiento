 6. CICLO COMPLETO: CREAR Y COMPLETAR UNA CITA
═══════════════════════════════════════════════════════════════════════════════

DÍA 1 - ADMIN CREA CITA:
──────────────────────

1. Admin va a citas.php
2. Click en "Nueva Cita"
3. Modal aparece
4. Ingresa datos:
   - Cliente: "Juan Pérez"
   - Teléfono: "+56912345678"
   - Email: "juan@email.com"
   - Dirección: "Calle Principal 123"
   - Tipo: "Instalación"
   - Fecha: "2026-05-15"
   - Hora: "10:00"
   - Técnico: "Freddy Morales"
   - Descripción: "Instalación de fibra óptica"
5. Click "Guardar"
6. JavaScript POST → citas.api.php
7. Validar: fecha no pasada, hora dentro rango, sin conflictos
8. Si OK → CitaController.php → Cita.php → INSERT en BD
9. Respuesta: {success: true}
10. Tabla se recarga, cita aparece
11. Estado: "PENDIENTE" (verde)


DÍA SIGUIENTE - TÉCNICO VE LA CITA:
────────────────────────────────────

1. Freddy se autentica
2. Va a calendario.php
3. Ve la cita en viernes 10:00
4. Color: VERDE (instalación)
5. Click en cita
6. Modal muestra detalles
7. Click "Marcar como Completada"
8. Textarea solicita observaciones
9. Freddy ingresa: "Instalación exitosa, velocidad OK"
10. Click "Guardar"
11. POST → citas.api.php (action=markCompleted)
12. Verificar que Freddy es técnico asignado
13. UPDATE citas SET estado='completada'
14. INSERT en historial_citas
15. Respuesta: {success: true}
16. Tabla/calendario se actualiza
17. Estado: "COMPLETADA" (gris)


ADMIN VE ESTADÍSTICAS:
──────────────────────

1. Admin va a dashboard.php
2. Panel principal muestra:
   - Total citas: 1
   - Completadas: 1
   - Pendientes: 0
   - Por tipo: Instalaciones 1
3. Tabla de semana muestra la cita como COMPLETADA
4. Admin puede hacer click para ver observaciones

 10. FLUJO DE INSTALACIÓN
═══════════════════════════════════════════════════════════════════════════════

PASO 1: Crear carpeta
─────────────────────

    cPanel → Folder Manager
            → public_html
            → Crear "sistema_agendamiento"


PASO 2: Subir archivos
──────────────────────

    FTP o cPanel File Manager
    ↓
    Subir toda carpeta "sistema_agendamiento"
    ↓
    Estructura automática se crea


PASO 3: Crear base de datos
───────────────────────────

    cPanel → MySQL Databases
            → Crear DB "internet_agenda"
            → Crear usuario "internet_agenda_user"
            → Password: (segura)
            → Asignar privilegios


PASO 4: Importar SQL
────────────────────

    phpMyAdmin
    ↓
    Seleccionar BD
    ↓
    Ir a "Importar"
    ↓
    Subir database/init.sql
    ↓
    Ejecutar
    ↓
    Tablas creadas ✓
    Datos insertados ✓


PASO 5: Configurar config.php
──────────────────────────────

    Editar: config/config.php
    ↓
    Cambiar:
    - DB_HOST: "localhost"
    - DB_USER: "internet_agenda_user"
    - DB_PASS: "contraseña_real"
    - DB_NAME: "internet_agenda"
    - APP_URL: "https://tu-dominio.com/sistema_agendamiento/"
    ↓
    Guardar


PASO 6: Acceder al sistema
──────────────────────────

    Navegador:
    https://tu-dominio.com/sistema_agendamiento/
    ↓
    Debe aparecer login
    ↓
    Login OK ✓

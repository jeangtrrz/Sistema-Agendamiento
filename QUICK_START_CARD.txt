🚨 PROBLEMAS RÁPIDOS

"No puedo entrar"
→ Verificar URL y credenciales
→ Ejecutar verificar.php
→ Revisar config/config.php

"Las citas no aparecen"
→ Verificar fecha sea esta semana
→ Limpiar caché (Ctrl+Shift+Delete)
→ Revisar logs/error.log

"Olvidé contraseña"
→ Contactar admin
→ Admin puede resetear en phpMyAdmin

MANUAL RÁPIDO DE ACCIONES

CREAR CITA (Solo Admin):
→ Citas → Nueva Cita
→ Llenar formulario
→ Seleccionar técnico
→ Guardar

VER CITAS (Admin + Técnico):
→ Citas (todo lo asignado)
→ O Calendario (vista semanal)

COMPLETAR CITA (Técnico):
→ Citas → Buscar cita asignada
→ Click "Completar"
→ Agregar observaciones
→ Guardar

VER REPORTES (Admin + Técnico):
→ Reportes
→ Seleccionar rango de fechas
→ Generar reporte
→ Ver estadísticas

CAMBIAR CONTRASEÑA (Todos):
→ Perfil (esquina superior)
→ Cambiar Contraseña
→ Ingresar actual y nueva
→ Guardar

 📌 OBJETIVO DEL PROYECTO
═══════════════════════════════════════════════════════════════════════════════

Crear una plataforma web moderna para la gestión centralizada de citas de:
  • Instalaciones de servicios de internet
  • Retiro de equipamiento
  • Visitas de soporte técnico

Permite que el equipo técnico y administrativo:
  ✓ Registre nuevas citas
  ✓ Visualice el calendario semanal
  ✓ Marque citas como completadas
  ✓ Genere reportes de actividad
  ✓ Gestione permisos por rol

═══════════════════════════════════════════════════════════════════════════════
 ⚙️ CONFIGURACIÓN Y CUSTOMIZACIÓN
═══════════════════════════════════════════════════════════════════════════════

Para cambiar cualquier cosa, edita config/config.php:

HORARIO OPERACIONAL:
  define('HORA_INICIO', '09:00');  // Cambiar a tu horario
  define('HORA_FIN', '18:00');

DURACIÓN CITA:
  define('DURACION_CITA_MINUTOS', 60);  // Por defecto 60, cambiar si necesario

TIPOS DE CITA:
  $GLOBALS['TIPOS_CITA'] = ['instalacion', 'retiro', 'soporte'];
  // Agregar/eliminar tipos según necesites

COLORES:
  $GLOBALS['COLORES_CITA'] = [
      'instalacion' => '#4CAF50',  // Verde
      'retiro' => '#FF9800',       // Naranja
      'soporte' => '#2196F3'       // Azul
  ];
  // Personalizar colores hexadecimales

DOMINIO:
  define('APP_URL', 'https://tu-dominio.com/sistema_agendamiento/');
  // Cambiar a tu dominio real

═══════════════════════════════════════════════════════════════════════════════
 🆘 SOPORTE Y TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: No puedo acceder al sistema
SOLUCIÓN: Revisar INSTALACION.md sección "Troubleshooting"

PROBLEMA: Base de datos no conecta
SOLUCIÓN: 1) Verificar credenciales en config.php
          2) Ejecutar verificar.php
          3) Revisar logs/error.log

PROBLEMA: Olvidé contraseña
SOLUCIÓN: 1) cPanel → phpMyAdmin
          2) Tabla usuarios → Editar
          3) Campo password = hash de "123456"
          4) Ahora puedes acceder con 123456

PROBLEMA: Técnico puede ver citas que no le asignaron
SOLUCIÓN: No debería pasar, revisar logs/error.log

PROBLEMA: Citas no aparecen en calendario
SOLUCIÓN: 1) Verificar fecha sea esta semana
          2) Limpiar caché navegador (Ctrl+Shift+Delete)
          3) Revisar logs/error.log

Para problemas más complejos:
→ Revisar documentación técnica
→ Contactar al desarrollador
→ Revisar código fuente con comentarios
